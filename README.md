## ✍️ Creating a New Post

Create a new file in `_posts/` with the format:

```
YYYY-MM-DD-title-of-post.md
```

Use the following front matter:

```yaml
---
layout: post
title: "Your Post Title"
date: YYYY-MM-DD HH:MM:SS
categories: [category1, category2]
tags: [tag1, tag2]
---

Your content here...
```
